# Screenshots

This folder contains visual proof of the Jenkins CI server setup on AWS EC2.

## Recommended Screenshots

To showcase your DevOps work, consider adding screenshots of:

1. **EC2 Instance Running**
   - AWS Console showing instance in "running" state
   - Instance details (Public IP, Security Groups, etc.)
   - File: `ec2-instance-running.png`

2. **SSH Connection Established**
   - Terminal showing successful SSH connection
   - Example: `ubuntu@ip-172-31-xx-xx:~$`
   - File: `ssh-access.png`

3. **Jenkins Unlock Screen**
   - Browser showing Jenkins initial unlock page
   - File: `jenkins-unlock.png`

4. **Jenkins Dashboard**
   - Jenkins main dashboard after successful setup
   - File: `jenkins-dashboard.png`

5. **Service Status**
   - Terminal output of `sudo systemctl status jenkins`
   - File: `jenkins-service-status.png`

## Privacy Note

⚠️ **Before committing screenshots:**
- Blur or redact public IP addresses if desired
- Remove any sensitive information
- Ensure no AWS account IDs or credentials are visible
